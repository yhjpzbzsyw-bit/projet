# 🎁 Diamas Création

## 📝 Description

**Diamas Création** est un site vitrine présentant la marque personnalisée dans des  **magazines personnalisés**, des **tableaux personnalisés** et des **box cadeaux**, conçus pour offrir des créations uniques et pleines de sens à ses clients.

## ✨ Sections du site

- **Accueil / Introduction** — présentation générale de la marque et de son univers
- **Pourquoi ce projet ?** — l'histoire et la motivation derrière DIAMA'S Création
- **Nos services** — présentation des prestations proposées par la marque
- **Nos créations** — les différents types de produits (magazines personnalisés, tableaux personnalisés, box cadeaux)
- **Quiz** — un quiz interactif pour aider les visiteurs à trouver la création qui leur correspond
- **Devis** — une section permettant de demander un devis personnalisé
- **Contact** — un moyen de contacter facilement la marque pour toute question ou commande

## 🛠️ Technologies utilisées

- **HTML5** — structure des pages
- **CSS3** — mise en forme personnalisée
- **Tailwind CSS** — framework CSS utilitaire pour un design rapide et cohérent
- **JavaScript** — interactions et animations (quiz, formulaire de devis, etc.)
- **Git & GitHub** — gestion de version et hébergement du code

## 📂 Structure du projet

```
├── index.html          # Page d'accueil
├── style.css            # Feuille de style personnalisée
├── script.js            # Scripts JavaScript
└── images/               # Visuels des créations (box, cadeaux, magazines...)
```
